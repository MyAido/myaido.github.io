Custom Pre-prompts

Create your own AI commands using the Pre-prompts feature. specific to your needs.

Example Use Cases:

  • Translator: "Translate text to Spanish"
  • Code Fixer: "Fix bugs in this Java code"
  • Summarizer: "Summarize in 3 bullet points"

How it works

  1. Go to Aido Settings > Pre-prompts
  2. Create a new trigger (e.g., "Summ")
  3. Add instructions: "Summarize this text"
  4. Select text in any app & choose "Summ"!
